# Mandatory Rules

Before implementation the AI shall:

- read all required documents;
- analyse the current repository;
- identify the next logical task;
- verify that no architectural blocker exists.

Implementation shall not begin before the required analysis has been completed.

---

# Repository First

The repository is the primary source of truth.

Conversation history must never replace repository analysis.

---

# Expected Output

Unless working directly inside the repository, every completed sprint shall provide:

- modified files;
- complete contents of new files;
- exact modifications or complete replacement of existing files;
- documentation updates;
- commit message.
