# Official Sandero Configurator Evidence Review

Date: 2026-07-23

## Decision

Accept the official Dacia Poland Sandero version pages as configuration-level evidence for four Eco-G 120 catalogue-price states and for the short equipment highlights shown on those same version pages.

Add the two explicitly listed automatic states to the active catalogue. Do not treat a broad equipment page or the default Essential TCe 100 configurator state as proof that an item applies to Eco-G.

## Exact accepted states

- Expression — Eco-G 120 manual;
- Expression — Eco-G 120 automatic;
- Journey — Eco-G 120 manual;
- Journey — Eco-G 120 automatic.

Each accepted state retains its observation date, grade code, exact powertrain label, gearbox and catalogue price.

## Evidence boundary

A version highlight is expanded only when the same official version page explicitly lists the target Eco-G configuration. The highlight list is not a complete equipment specification, so missing items do not create negative availability observations.

The broad equipment pages mix standard equipment, packages and potentially configuration-dependent entries. Their named packages remain supporting evidence only until an exact configuration state proves applicability, content and price.

## Default configurator state

The default visible configurator state is Essential TCe 100 manual. The zero-price driver-seat-height adjustment and split rear-seat observations remain in the snapshot but create no Eco-G master-data rows.

## Reporting consequence

The two automatic configurations form a new `sandero_ecog120_automatic` scope. The complete release now contains 69 active configurations and 18 independent scopes.

## Follow-up

Repeat the same exact-state process for Jogger. Additional Sandero packages or options may be imported later only when a stable deep link or another official source proves the exact trim, engine, gearbox, item content and price.
