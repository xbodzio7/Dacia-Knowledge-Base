# Official Sandero Stepway Configurator Evidence Review

Date: 2026-07-23

## Decision

Accept the official Dacia Poland Sandero Stepway version pages as configuration-level evidence for the five active Eco-G 120 price states and for the short equipment highlights presented on those same version pages.

Do not treat the default configurator as proof that an option or package applies to another engine, gearbox or trim.

## Exact accepted states

- Essential — Eco-G 120 manual;
- Expression — Eco-G 120 manual;
- Expression — Eco-G 120 automatic;
- Extreme — Eco-G 120 manual;
- Extreme — Eco-G 120 automatic.

Each accepted state retains its observation date, version grade code, exact powertrain label, gearbox and catalogue price.

## Evidence boundary

The broad equipment pages are retained as supporting evidence but are not flattened blindly. They may contain technical qualifiers or entries whose applicability depends on engine, gearbox, regulatory state or another selected option.

A version highlight is expanded only when the same official version page explicitly lists the target Eco-G configuration. Absence from the highlight list is not interpreted as `not_available`.

## Default configurator package

The default visible configurator state is TCe 110 manual and shows `pakiet MEDIA DISPLAY` for 1,900 PLN. Because the page does not prove Eco-G applicability, the observation remains only in the snapshot and creates no commercial-item or configuration mapping.

## Follow-up

Repeat the same exact-state process for the remaining models, beginning with Sandero. A future package may import additional Stepway packages only after a stable deep link or another official source proves the specific trim, powertrain, gearbox, content and price combination.
