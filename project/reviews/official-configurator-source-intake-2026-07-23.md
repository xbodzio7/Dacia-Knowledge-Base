# Official Dacia Configurator Source Intake Review

Date: 2026-07-23

## Decision

Register official Polish Dacia configurators as volatile, dated web evidence and use them as a first-choice source for current standard equipment, packages, individual options, prices, dependencies and exclusions.

## Evidence registered

- official configurator catalogue;
- Sandero configurator;
- Sandero Stepway configurator;
- Duster configurator;
- Jogger configurator;
- Bigster configurator;
- Stepway trim-equipment pages for Essential, Expression and Extreme.

## Boundaries

A dynamic configurator is not an immutable price-list document. Every observation must retain its date, exact model, trim, powertrain and gearbox state. A package visible on one default configuration does not prove applicability elsewhere.

The first intake therefore registers URLs and directly visible observations only. It does not create new commercial-item mappings. A later package must capture configuration states individually and cross-check them against the effective official PDF price list before importing master data.

## Images

Official photographs are not copied into the repository or release assets. Until redistribution rights and stable official asset URLs are established, the offline product uses original deterministic silhouettes that distinguish model body and trim treatment.
