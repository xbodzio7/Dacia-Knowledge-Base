## Browser verification

The exact HTML extracted from the release archive was exercised in Chromium. It started with 67 configurations. Selecting Jogger retained 22 results. Three configurations produced 59 full equipment comparison rows, and `PokaĹĽ tylko rĂłĹĽnice` reduced the visible comparison to 15 rows. Three comparison silhouettes and five model-picker silhouettes were present. The technical configuration code was not visible and no JavaScript console errors occurred.

The silhouettes are repository-generated visual identifiers, not official vehicle photographs and not exact representations of paint, wheels or upholstery.

## Official configurator boundary

The official Polish configurators are registered as dated, dynamic web sources. Visible options and prices are not copied across trims, powertrains or gearboxes without explicit configuration-level evidence. This release records the source intake but does not yet claim a complete option catalogue for every configuration.
